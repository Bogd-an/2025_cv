### **Опис коду**

Наданий скрипт реалізує ефект мозаїки для статичного зображення `kpi.jpg`.

```python
import cv2

WINDOW_NAME = 'Mosaic Effect'
IMAGE_PATH = 'kpi.jpg'

img = cv2.imread(IMAGE_PATH)

def apply_mosaic_effect(trackbar_value):
    scale = max(trackbar_value, 1) / 100.0
    h, w = img.shape[:2]
    small = cv2.resize(img, (0, 0), fx=scale, fy=scale, interpolation=cv2.INTER_LINEAR)
    mosaic = cv2.resize(small, (w, h), interpolation=cv2.INTER_NEAREST)
    cv2.imshow(WINDOW_NAME, mosaic)

cv2.namedWindow(WINDOW_NAME)
cv2.createTrackbar('Scale', WINDOW_NAME, 1, 100, apply_mosaic_effect)

cv2.waitKey()
```

Алгоритм працює наступним чином: зображення спочатку стискається до меншого розміру (`small`) з коефіцієнтом `scale`, що контролюється повзунком. Потім це маленьке зображення розтягується до початкового розміру з використанням методу інтерполяції `INTER_NEAREST`, що створює чіткі квадратні "пікселі", формуючи ефект мозаїки.


### **Результат виконання**

У результаті виконання програми відкривається вікно, в якому відображається зображення. За допомогою повзунка "Scale" можна в реальному часі керувати ступенем пікселізації: менше значення на повзунку відповідає сильнішому ефекту мозаїки, а більше — слабшому.
